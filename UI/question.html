How do I recursively return an integer from a sigmoid function?

I am new to Game programming. We are working on a project that sort of needs a real time simulation of a snake movement. I have heard of the sigmoid function, how do I use this function to return an integer, so I can take some actions based on its movement on the screen?

A sigmoid function is a mathematical function having a characteristic "S"-shaped curve or sigmoid curve. Often, sigmoid function refers to the special case of the logistic function shown in the first figure and defined by the formula

1/(1+e^(-x))

Obtaining finite values from this function is possible , but be made known that the only two stable finite integers that can be obtained form the sigmoid function as output are 1 and -1. It will approach this output once its input approaches infinity. In other words, The larger the value you feed in to the sigmoid function, the more stable it becomes, and it tends to approach 1 or -1. This function is extensively used in games, animations to mimic trying to push objects at rest, the stubborn behaviour exibited by this object formally at rest to resist pull or push to its new position is what this function carefully mimics.
while (y < 1) { let i = 0; let y = 1/(1+Math.exp(-i)); i+= 1; } console.log(y);
